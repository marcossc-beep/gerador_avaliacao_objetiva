MSEP SC - Layout Architect

Instruções de Operação MSEP (Preto e Branco Otimizado):

  1. Cole o arquivo JSON estruturado contendo as chaves metodológicas (capacidade, contextualizacao e comando).
  2. O interpretador montará os blocos seguindo rigorosamente a segmentação pedagógica exigida pela instituição.
  3. O algoritmo de preenchimento inteligente de lacunas continua ativo para compactar itens P, M e G visando economia de folhas.
  4. Clique em "Gerar Layout de Avaliação" para disparar a visualização e o comando de impressão.
